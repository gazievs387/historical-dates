# Запуск Проекта

Для запуска проекта выполните следующие шаги:

1.  **Установка зависимостей:**
    ```bash
    npm install
    ```

2.  **Запуск приложения в режиме разработки:**
    ```bash
    npm run dev
    ```

Приложение будет доступно по адресу **http://localhost:3000** (на 3000 порту).
